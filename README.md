## CS2死后全自动狡辩工具
用法：
* 以管理员身份打开后进入游戏即可。

* 第一次运行会在同级目录生成打开聊天键设置KeyConfig.json以及狡辩语料Messages.json，您可以自行修改其中内容。

须知：
* 程序调用WindowsAPI，不放心的可以自行编译使用。
* 内容输入逻辑为将随机语料添加到剪贴板进行粘贴，此程序会污染你的剪贴板。
